and outputs. This means XIOFileChecker compares what should be written by
`XIOS <https://forge.ipsl.jussieu.fr/ioserver/>`_ according to the
`DR2XML <https://github.com/senesis/dr2pub>`_ files definition and what
was really written on disk.
